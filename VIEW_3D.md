## ğŸ› ï¸ Troubleshooting

**Server won't start?**

- Make sure port 8000 is available
- Try: `python3 server.py` (should show the URL)

**Scene doesn't load?**

- Check that your JSON file is valid: `python3 -m json.tool your_file.json`
- Make sure the file is in the same directory

**Browser shows blank?**

- Check browser console for errors (F12)
- Make sure JavaScript is enabled
- Try a different browser (Chrome/Firefox recommended)

**Can't see the scene?**

- Click "Reset View" button
- Try zooming out (scroll wheel)
- Check that the JSON has rooms, walls, etc.
